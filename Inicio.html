What is network security?

Network security is a category of practices and technologies that keep internal networks protected from attacks and data leaks. It includes access control, cyber attack prevention, malware detection, and other security measures.

NETWORK AUDIT

Internal network auditing is one that reviews the protocols used as well as the configuration elements. For its part, the external would be the one focused on analyzing access elements such as firewalls, routers, or IDS/IPS configurations. Wireless or wired.

What is WEP?

WEP encrypts traffic with a 64-bit or 128-bit hexadecimal key. This is a static key, which means that all traffic is encrypted with a single key, regardless of device. A WEP key allows computers on a network to exchange encrypted messages while hiding the content of the messages from intruders. This key is the one used to connect to a network with wireless security enabled.

What is WPA?

Later, WPA or Wi-Fi Protected Access emerged. This protocol was the Wi-Fi Alliance's replacement for WEP, which was integrated in 2003. It shared similarities with WEP, but offered improvements in how it handled security keys and how users were authorized. While WEP provides the same key to every authorized system, WPA uses the Temporal Key Integrity Protocol (TKIP), which dynamically changes the key used by systems. This prevents intruders from creating their own encryption key to match the one used by the protected network. 

What is WPA2?

WPA2 was introduced in 2004 and was an enhanced version of WPA. WPA2 is based on the Robust Security Network (RSN) mechanism and works in two modes:• Personal Mode or Pre-Shared Key (WPA2-PSK): It is based on a shared access code and is generally used in home environments.• Enterprise Mode (WPA2-EAP): As the name suggests, this mode is more suitable for use in companies or organizations.Both modes use CCMP, which stands for Counter Mode Cipher Block Chaining Message Authentication Code Protocol. 

BEINI

Basically, Beini is an application or distribution in charge of analyzing Wi-Fi networks. With analysis, we mean that it was created for network security.
This utility has tools that will facilitate the task, such as Feedingbottle, Aircrack-ng or Minidwep-gtk.

BackTrack

BackTrack was a GNU/Linux distribution in LiveCD format thought and designed for general computer security auditing. Its successor is the Kali Linux distribution which was developed from the rewrite of this distribution by the same Offensive Security team.
